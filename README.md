# Fantasy Analytics

Fantasy sports analytics dashboard. Built with Vite + React + TypeScript, Express, Clerk auth, Redux Toolkit, TanStack Query, and Tailwind v4.

## Getting started

### Client
```bash
cd client
npm install
cp .env.example .env  # add your VITE_CLERK_PUBLISHABLE_KEY
npm run dev
```

### Server
```bash
cd server
npm install
# add CLERK_SECRET_KEY to server/.env
npm run dev
```
